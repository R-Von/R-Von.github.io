新建Larvel项目并启动

服务器工具

使用 laragon-wamp

laragon

composer 默认外国镜像 所以导致 下载 和安装都非常慢 或 翻墙
配置 composer 使用中国镜像 [https://pkg.phpcomposer.com/]

composer config -gl repo.packagist
内容如下
    [repositories.packagist.org.type] composer
    [repositories.packagist.org.url] https?://packagist.org

composer config -g repo.packagist composer https://packagist.phpcomposer.com
内容如下
    [repositories.packagist.org.type] composer
    [repositories.packagist.org.url] https://packagist.phpcomposer.com

安装larvel框架

安装(生成项目 项目初始化)

composer create-project --prefer-dist laravel/laravel [自定义名字(不是必填)]

本地开发服务器

如果你在本地安装了 PHP，并且想要使用 PHP 内置的开发环境服务器为应用提供服务，可以使用 Artisan 命令 serve：

php artisan serve

该命令将会在本地启动开发环境服务器，这样在浏览器中通过 http://localhost:8000 即可访问应用
